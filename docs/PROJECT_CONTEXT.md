## Proyecto Actual

Nombre:

LeadPulse AI

Descripción:

Aplicación web SaaS con un agente IA para prospección comercial B2B.

El usuario introduce una descripción de su cliente ideal (ICP).

El agente:

* Analiza el perfil.
* Busca prospectos.
* Los califica.
* Genera un correo personalizado.
