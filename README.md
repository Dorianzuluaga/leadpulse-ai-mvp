# LeadPulse AI - MVP Product Requirements

## Descripción

LeadPulse AI es una aplicación web SaaS conceptual diseñada para ayudar a equipos comerciales B2B a reducir el tiempo invertido en la búsqueda y preparación de prospectos potenciales.

El proyecto explora cómo la Inteligencia Artificial puede asistir en la prospección comercial mediante la identificación, calificación y preparación de leads a partir de una descripción del cliente ideal.

Este repositorio forma parte del proyecto **"Creando los Requisitos de tu Producto con IA"** y se centra en la definición del producto antes de su desarrollo técnico.

---

## Problema

Los equipos comerciales dedican una cantidad significativa de tiempo a:

* Buscar prospectos potenciales.
* Investigar información de contacto.
* Analizar oportunidades comerciales.
* Redactar mensajes personalizados.

LeadPulse AI busca reducir este esfuerzo mediante un agente de IA capaz de asistir en estas tareas.

---

## Objetivo del MVP

Permitir que un usuario:

1. Defina un perfil de cliente ideal.
2. Obtenga una lista de prospectos potenciales.
3. Visualice información relevante de cada prospecto.
4. Reciba una calificación de oportunidad.
5. Genere un correo personalizado para iniciar el contacto.

---

## Flujo General

```text
Usuario introduce ICP
        ↓
Agente analiza perfil
        ↓
Sistema identifica prospectos
        ↓
Agente califica oportunidades
        ↓
Generación de email personalizado
```

---

## Alcance del MVP

### Incluido

* Definición de cliente ideal (ICP)
* Búsqueda de prospectos
* Calificación de leads
* Verificación humana/aprobación
* Generación de correos personalizados
* Interfaz sencilla

### Excluido

* CRM
* Login y gestión de usuarios
* Integraciones externas
* Redes sociales
* Automatizaciones complejas
* Pagos y suscripciones
